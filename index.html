Easy Installation

Boxly isn't yet on the Chrome Web Store, but you can easily install it with these steps:

1

Download Boxly

Download or clone the Boxly project and unzip it to a convenient location on your computer.

2

Open Chrome Extensions

Launch Chrome and click the Extensions button in the top-right corner, then select "Manage Extensions".

3

Enable Developer Mode

Toggle "Developer Mode" on in the top-right corner of the Extensions page.

Developer mode
4

Load Unpacked Extension

Click "Load Unpacked" and navigate to select the Boxly project folder.

5

Start Using Boxly!

Enable Boxly and click the extension icon to open your productivity dashboard.
