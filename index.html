How Boxly Works

1

Create Your Tasks

Add tasks using our intuitive sticky note interface. Set due dates, topics, and estimated completion time.

2

Start Working

Click start on any task to activate the AI-powered blocking system and begin your focused work session.

3

Stay Focused

Boxly's Llama3 AI analyzes websites you visit and blocks distracting content while allowing relevant resources.

4

Take Smart Breaks

Get automatic break reminders to maintain productivity and prevent burnout during long work sessions.

Easy Installation

Boxly isn't yet on the Chrome Web Store, but you can easily install it with these steps:

1

Download Boxly

Download or clone the Boxly project and unzip it to a convenient location on your computer.

2

Open Chrome Extensions

Launch Chrome and click the Extensions button in the top-right corner, then select "Manage Extensions".

3

Enable Developer Mode

Toggle "Developer Mode" on in the top-right corner of the Extensions page.

Developer mode
4

Load Unpacked Extension

Click "Load Unpacked" and navigate to select the Boxly project folder.

5

Start Using Boxly!

Enable Boxly and click the extension icon to open your productivity dashboard.
